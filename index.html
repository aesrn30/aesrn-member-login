System Information

About This System

Owner: African Emerging Scholars Research Network (AESRN)

Purpose: Secure portal for authorized members to submit research information and collaborate within the network. This system facilitates the collection and management of research data from emerging scholars across Africa.

Access Level: Restricted to members only.

Security Features

  • Auto-logout: Session automatically expires after 5 minutes of inactivity
  • Session Management: Secure session persistence across page reloads
  • Single Session: Users cannot login simultaneously from multiple devices
  • Brute Force Protection: Temporary lockout after multiple failed login attempts
  • Credential Validation: Secure server-side validation of user credentials
  • Encrypted Storage: Local session data encryption
  • Activity Tracking: Real-time monitoring of user activity

Technical Specifications

  • Technology Stack: HTML5, CSS3, Vanilla JavaScript
  • Responsive Design: Fully optimized for desktop, tablet, and mobile devices
  • Data Storage: LocalStorage for session management with expiry
  • Integration: Tally.so for form data collection and management
  • Cross-Browser: Compatible with Chrome, Firefox, Safari, Edge
  • Performance: Optimized for fast loading and smooth interactions
  • Time Zone: Ghana Time (GMT) displayed in footer

User Management

Access is granted to pre-approved email addresses. The system supports:

  • Individual user authentication
  • Single session per user (no simultaneous logins)
  • Session persistence for uninterrupted workflow
  • Secure logout with confirmation
  • Activity timeout warnings
  • Dark/light mode preferences
  • Personalized login greetings
